# Emergency-hospital-waiting-time-prediction
Project Overview
This project predicts hospital emergency department waiting time using machine learning models.
It helps estimate how long a patient may wait based on hospital conditions like queue length, triage level, and staff availability.

Dataset
The dataset contains hospital emergency records with features like:

Patient Age
Gender Code
Arrival Time
Triage Level
Queue Length
Doctors Available
Nurses Available
Emergency Type
Patient Wait Time (target)
Models Used
Linear Regression
Random Forest
Support Vector Regression (SVR)
XGBoost
CatBoost
